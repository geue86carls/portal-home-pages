# portal-home-pages

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态页面的集中存放和版本管理使用。

## 项目简介

本仓库主要存放各类独立的 HTML 页面文件。这些页面可以是导航页、信息页、临时页面或其他静态内容。仓库本身不提供任何在线服务或动态功能，所有页面均以纯静态 HTML 形式保存，便于查看、分享或进一步开发。

## 目录说明

```
portal-home-pages/
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 页面引用的静态资源（CSS、JS、图片等）
└── README.md        # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立页面，文件名通常反映页面用途或版本。
- `assets/`：按需存放页面所需的样式、脚本或图像资源。

## 页面归档说明

- 所有页面文件均为独立 HTML，不依赖外部数据库或后端。
- 每个页面尽可能自包含，资源引用路径相对于仓库内部。
- 页面内容以归档和展示为目的，不包含任何商业或推广性质的内容。

## 维护说明

- 欢迎提交 Pull Request 添加新的页面或改进已有页面。
- 请确保新增页面为独立 HTML，且不包含任何外部敏感链接或恶意代码。
- 对于较大的改动，建议先在 Issue 中讨论。
- 仓库维护者会定期检查内容，确保页面可正常访问和渲染。
